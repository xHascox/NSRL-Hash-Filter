Dieses Programm erlaubt es einem, das NSRL Hast Set nach gewissen Attributen zu filtern. Zum Beispiel kann man nur die Hashwerte von Antivirenprogrammen exportieren, oder nach einem bestimmten Hersteller, Dateiendung, Betriebssystem oder Produkt(-Typ) filtern. 

Wie man vorgeht:
1. Den Knopf Select .iso drücken
2. RDS_modern.iso Datei auswählen (erhältlich auf der NIST Webseite https://www.nist.gov/itl/ssd/software-quality-group/national-software-reference-library-nsrl/nsrl-download/current-rds)
3. Den Knopf Extract .iso drücken
4. Warten (<1 Minute)
5. Die gewünschten Filter eingeben:
	Zu beachten:
		Ausser bei "File Extension" immer einen Vorschlag im Dropdown Menu auswählen, denn:
		Nur exakte Übereinstimmungen gelten.
	Tipp: Mehrfachauswahlen sind mit Kommas getrennt möglich
	Tipp: Keine Eingabe oder ein Komma am Ende = Wildcard
6. Auf Export for ... drücken.
7. Warten (~ 1 Stunde)

Die Version _LowRam ist etwas langsamer als die normale version, benötigt jedoch nur 1 GB Ram und lässt sich somit auch auf Laptops ausführen.
